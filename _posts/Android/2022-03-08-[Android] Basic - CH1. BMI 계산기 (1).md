### BMI 계산법

공식 = 몸무게(kg) / (키(m) * 키(m))


|    분류    |    수치     |
| :--------: | :---------: |
|   저체중   |   < 18.5    |
|    정상    | 18.5 ~ 22.9 |
|   과체중   |  23.0 이상  |
|  위험체중  | 23.0 ~ 24.9 |
| 1단계 비만 | 25.0 ~ 29.9 |
| 2단계 비만 | 30.0 ~ 34.9 |
|  고도비만  |  35.0 이상  |
